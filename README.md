## 🎯 Tic Tac Toe

A simple Tic Tac Toe game written in Java using a graphical user interface (GUI).

It allows two players to take turns placing “X” and “O” and win by getting three symbols in a row (horizontally, vertically, or diagonally).

## 💻 Technologies

• Java

## 🚀 Features

•	Graphical user interface (GUI)

•	Two-player mode

• AI with difficulty mode

•	Interactive game board

•	Win detection

•	Draw detection

## 🚦 How to Run Tic Tac Toe

• Clone the repository: git clone https://github.com/MarvinWod/Tic-Tac-Toe-Java-v1
  
• Open the project in your Java IDE.

• Run the main class.
